# Student Management System (Java + MySQL)

This is a simple Student Management System developed using Java and MySQL.
It allows users to add, update, delete, and view student records.
This project is useful for managing student information in a database.

## Features
- Add new student
- View student details
- Update student information
- Delete student record
- Store data using MySQL database

 ## Technologies Used
- Java
- MySQL
- JDBC
- MySQL Workbench

## Project Structure
StudentManagementSystem/
│
├── src/
│   └── Student.java
│   └── Main.java
│
├── database/
│   └── student.sql
│
└── README.md


## How to Run

1. Clone the repository
git clone https://github.com/bhushan307x/StudentManagementSystem.git
2. Open project in IDE (VS Code)
3. Import MySQL database using student.sql
4. Run the Main.java file


## Output
Program successfully performs CRUD operations on student data.
